记录

刀剑神域无星之夜的咏叹调采用了HEVC main 10的编码

启用HEVC 10-bit解码后：

  • jellyfin使用的ffmpeg命令略有不同：
    • 从-vaapi_device /dev/dri/renderD128变为-hwaccel vaapi -hwaccel_output_format vaapi -vaapi_device /dev/dri/renderD128
  • 显示的转码帧数降低了
  • 改变jellyfin转码设置，是立即生效的，无需重启docker
    • GPU编码时
  • 使用CPU编码，会卡很久才能播放
    • CPU编码显示-codec:v:0 libx264，GPU编码显示h264_vaapi
  • 实测RX550的薄弱GPU性能，勾选HEVC 10bit解码，反而会拖累编码过程，帧数更低。

CPU编解码: h265 -> h264

GPU编解码：h265 -> GPU -> 中间帧 -> GPU -> h254

GPU编码，CPU解码：h265 10bit -> CPU -> 中间帧 -> GPU -> h254
